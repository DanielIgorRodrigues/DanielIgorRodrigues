Tenho mais de **13 anos de experiência em tecnologia**, passando por suporte, qualidade de software e desenvolvimento. Essa trajetória me deu uma visão bem prática sobre produto, usuário final, análise de problemas, documentação, investigação técnica e colaboração com diferentes áreas.

Nos últimos anos, venho direcionando minha carreira para desenvolvimento de software, com foco em:

- APIs e aplicações com **C#/.NET**

- Uso estratégico de IA para acelerar análise, desenvolvimento e documentação

Gosto de trabalhar em soluções reais, especialmente quando envolvem performance, estabilidade, integração entre sistemas e melhoria contínua.
